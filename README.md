# React Todo List

A simple **React** Todo List app built with **Vite**.  
It supports adding, completing, and deleting todos, with **localStorage persistence** so your list stays after refresh.

---

## Features

- Add new todos
- Mark todos as completed
- Delete todos
- Persist todos in browser storage (`localStorage`)
- Clean, minimal UI styling

---

## Tech Stack

- React (Function Components + Hooks)
- Vite
- JavaScript (JSX)
- CSS
